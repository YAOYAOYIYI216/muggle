# React + TypeScript + Vite + StoryBook

storybook 8.1.1 + React 18

- [图标库地址](https://fontawesome.com/v6/search?o=r&m=free)

## 1.本项目初始化

```bash
# 安装依赖
npm i

# 启动本地服务
npm run dev

# 生产环境-打包命令
npm run build:prod
```

## 2.使用方式(在其他项目使用该组件库)

```bash
# 安装依赖
npm i sbui-react

# 引入样式
import 'sbui-react/distLib/style.css'

# 使用组件
import { Icon } from 'sbui-react';
```

## 3.发包

```bash
# 登录npm，没有账号需要先注册
npm login

# 发布组件库
npm publish

```
